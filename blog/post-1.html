用 Three.js 做一个可旋转的 3D 虚拟服装展示

把一件衣服放进网页,让用户用鼠标自由旋转、缩放查看,是数字时尚最直观的展示方式之一。本文用 Three.js 搭一个最小可运行示例,不依赖任何打包工具,直接用 ES Module 即可。

1. 准备模型

先用 Blender 或 CLO 3D 导出 .glb(GLTF 的二进制版,体积小、无需额外贴图文件)。把它放到项目的 models/ 目录。

2. 场景、相机、渲染器

Three.js 的三件套:场景(Scene)、相机(PerspectiveCamera)、渲染器(WebGLRenderer)。把它们挂到一个容器 div 上,并让相机看向原点。

import * as THREE from 'three'; import { OrbitControls } from 'three/addons/controls/OrbitControls.js'; const scene = new THREE.Scene(); const camera = new THREE.PerspectiveCamera(45, 1, 0.1, 100); camera.position.set(0, 1, 3); const renderer = new THREE.WebGLRenderer({ antialias: true, alpha: true }); renderer.setSize(480, 480); document.getElementById('stage').appendChild(renderer.domElement);

3. 加载模型 + 轨道控制器

GLTFLoader 读入模型,OrbitControls 负责"拖拽旋转、滚轮缩放"。加载完成后把模型加入场景,并用一个辅助函数让相机自动框住物体。

const controls = new OrbitControls(camera, renderer.domElement); controls.enableDamping = true; new GLTFLoader().load('models/garment.glb', (g) => { scene.add(g.scene); fitCameraToObject(g.scene, camera, controls); }); function animate() { requestAnimationFrame(animate); controls.update(); renderer.render(scene, camera); } animate();

4. 打光与环境

服装材质对光很敏感。用 HemisphereLight 做环境光,加一盏 DirectionalLight 勾出褶皱阴影;若用 MeshStandardMaterial,建议再叠加一个基于图像的照明(IBL,如 RoomEnvironment),布料质感会真实很多。

小结

至此,一个可旋转、可缩放的 3D 服装展示就完成了。后续可以加自动旋转、换色、甚至布料物理模拟,把它做成产品级的虚拟试穿入口。下一篇会讲如何把模型批量、标准化地导出来喂给这里。
