用 Three.js 做一个可旋转的 3D 虚拟服装展示A Rotatable 3D Garment Viewer with Three.js

把一件服装放进网页里自由旋转,是数字时尚最直观的呈现方式之一。借助 Three.js,这件事比想象中简单:加载模型、加灯光、挂一个轨道控制器即可。 Putting a garment into a web page and letting it spin freely is one of the most intuitive forms of digital fashion. With Three.js it is simpler than it looks: load a model, add lights, attach an orbit controller.

最小可用代码Minimal Code

先准备一个 .gltf 或 .glb 服装模型(可从 Clo3D / Marvelous Designer 导出),然后用 GLTFLoader 加载,并用 OrbitControls 实现鼠标拖拽旋转。 Prepare a .gltf or .glb garment model (export from Clo3D / Marvelous Designer), load it with GLTFLoader, and enable drag-to-rotate with OrbitControls. 

import * as THREE from 'three';
import { GLTFLoader } from 'three/addons/loaders/GLTFLoader.js';
import { OrbitControls } from 'three/addons/controls/OrbitControls.js';

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(45, w/h, 0.1, 100);
camera.position.set(0, 1, 3);
const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(w, h); document.body.appendChild(renderer.domElement);

const controls = new OrbitControls(camera, renderer.domElement);
new GLTFLoader().load('garment.glb', g => scene.add(g.scene));

再加一点质感A Touch of Material

服装材质很吃灯光。给场景加一盏环境光加一盏方向光,并把模型材质设为略带粗糙度的 MeshStandardMaterial,布料的褶皱感会更真实。 Garments are light-hungry. Add an ambient light plus a directional light, and use a slightly rough MeshStandardMaterial so the fabric folds read more convincingly.

到此,一个可旋转的虚拟服装展示就完成了。后续可以叠加换色、换面料、甚至接动作捕捉做虚拟人试穿。 That's a rotatable virtual garment viewer. Next you can layer on color/fabric swaps, or even motion-capture-driven virtual try-on.
