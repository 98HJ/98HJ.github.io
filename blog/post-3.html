柔性压力传感器的标定与噪声处理

标定决定了一个传感器"说不说真话"。一块没标定的柔性压力片,读出来的数字既无单位也不可比。本文以压阻织物压力传感器为例,记录一次从桥式电路到漂移补偿的完整标定流程,并给出可直接用的噪声处理思路。

1. 为什么必须标定

压阻材料批次差异大、迟滞明显,同一压力下不同样品读数能差一倍。标定就是建立物理量(压力)↔ 读数(电压/ADC)的映射,让数据可解释、可跨器件比较。

2. 桥式电路与激励

单个压阻元件直接分压,温漂和线阻会污染信号。更稳的做法是用半桥/全桥:把传感元件与参考电阻组成惠斯通电桥,输出差分电压,抵消共模漂移。

Vout = Vexc * (R_sensor - R_ref) / (R_sensor + R_ref)

用恒流源或稳定基准给桥供电,比直接接 VCC 更抗电源波动。

3. 标定流程

准备一组已知压力的标准砝码(或商用测力计),逐步加载并记录:

  • 从 0 开始,按 5/10/20/50/100 kPa 逐级加载,每级稳定 2–3 秒再读。
  • 再逐级卸载,得到"加载—卸载"两条曲线,用以评估迟滞。
  • 每组重复 3 次取平均,降低随机误差。

4. 拟合映射

把"压力—读数"点画出来,低段近似线性,高段常饱和。两步拟合:

  • 先看 线性回归,看 R² 够不够;
  • 不够就用多项式(2–3 次)或分段线性,弯曲处更准。

下面用 Python 做一次二次多项式拟合:

import numpy as np x = np.array([0, 5, 10, 20, 50, 100]) # 已知压力 kPa y = np.array([12, 130, 240, 460, 880, 1200]) # 对应 ADC 读数 coef = np.polyfit(y, x, 2) # 以读数为自变量,反推压力 pressure = np.polyval(coef, raw_adc) # 之后实时换算

5. 噪声来自哪里

  • 热噪声/约翰逊噪声:电阻本身,可用平均抑制。
  • 接触噪声:织物与电极接触微动,表现为"沙沙"高频抖动。
  • 电磁干扰(EMI):附近电机、开关电源串入 50/100 Hz 及其谐波。
  • 运动伪影:不是真噪声,但是最容易被误当成信号的假象。

6. 滤波策略

  • 移动平均 / 指数平滑:最简单,压高频抖动,但会拖慢响应。
  • 中值滤波:对偶发尖刺(接触火花)特别有效,保留阶跃。
  • 带阻( notch ):专门抠掉 50 Hz 工频干扰。
  • 卡尔曼(lite):有模型时更优,但参数要调,新手先用前三种。

7. 漂移补偿

温升/出汗让零点慢慢漂。两个实用办法:

  • 上电归零:每次开机采一段静息值作基准减去。
  • 高通/去趋势:对慢漂做高通,保留快速压力变化(适合动作识别)。

小结

标定的目标不是"曲线完美",而是让后续算法拿到有单位、可比较、低漂移的信号。把桥路、标定表、滤波三件事做对,柔性传感的数据质量会立刻上一个台阶。
