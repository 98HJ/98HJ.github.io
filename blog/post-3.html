柔性压力传感器的标定与噪声处理

标定决定了一个传感器"说不说真话"。本文记录一次织物压力传感器标定的完整流程,以及如何处理实测中的噪声。

1. 为什么标定

柔性压阻材料非线性、有迟滞,同一压力两次读数可能不同。标定就是建立"电阻 / 电压 → 真实压力"的映射,否则原始数据没有物理意义。

2. 桥式电路

单点电阻变化太小,常用半桥或全桥(惠斯通电桥)把微小变化转成差分电压,再放大。

V_out = V_ex * (R_x / (R_x + R_ref) - 0.5) // 半桥近似

3. 标定流程

逐档加载已知质量(如 0 / 50 / 100 / 200 / 500 g),记录稳定读数;用多项式或分段线性拟合压力–电压曲线。建议每个点静置 3 秒等蠕变稳定。

4. 噪声与漂移处理

· 硬件层:电源去耦、屏蔽线、工频(50 Hz)陷波。
· 软件层:移动平均 / 低通滤波;对缓慢漂移做高通或周期性归零。
· 漂移补偿:记录空载基线,实时减去。

小结

标定不是一次性工作。材料会老化、温湿度会变,量产时每片都要单独标定,或至少做温度补偿。标得好,后面的算法才好做。
